## 📌 프로젝트 개요

Next.js 기반의 프론트엔드 프로젝트로, 회원가입/로그인 인증 및 게시판 CRUD 기능을 구현합니다.

---

## 🗓️ 개발 일정

### 3월 4일 ~ 3월 5일 - 프로젝트 세팅
- Next.js 프로젝트 초기 세팅
- GitHub 레포지토리 연동
- Tailwind CSS, Zustand 등 패키지 설치

### 3월 6일 ~ 3월 9일 - 뼈대 구현
- 회원가입 페이지 구현
- 로그인 페이지 구현
- JWT 기반 로그인 인증 구현
- 게시판 페이지 구현
- 글쓰기 페이지 구현

### 3월 10일 ~ 3월 11일 - CRUD 구현
- 게시글 작성 (Create)
- 게시글 조회 (Read)
- 게시글 수정 (Update)
- 게시글 삭제 (Delete)

### 3월 12일 - 마무리
- 버그 수정 및 최종 점검
- 배포

---

## 🛠️ 기술 스택

| 분류 | 기술 |
|------|------|
| Framework | Next.js |
| Styling | Tailwind CSS |
| 상태관리 | Zustand |
| 인증 | JWT |
| 패키지 매니저 | Bun |

---

## 📂 주요 기능

- ✅ 회원가입
- ✅ 로그인 / 로그아웃
- ✅ JWT 인증
- ✅ 게시판 목록 조회
- ✅ 게시글 작성 / 수정 / 삭제

---

## 🚀 시작하기

```bash
# 패키지 설치
bun install

# 개발 서버 실행
bun run dev
```
